This repository contains the source code for the paper:

**Multi-Relaxation Assisted Heuristic for Cardinality Constrained Portfolio Optimization Problem**

## Problem Description

The Cardinality Constrained Portfolio Optimization (CCPO) problem extends the classical mean-variance portfolio optimization by imposing a limit on the number of assets that can be included in a portfolio. 
This makes the problem NP-hard in general. Our approach proposes a Multi-Relaxation Assisted Heuristic Method to efficiently find high-quality solutions.

## Requirements

- Python >= 3.8
- NumPy
- Pandas
- DOcplex (IBM Decision Optimization CPLEX Modeling for Python)
- Matplotlib
